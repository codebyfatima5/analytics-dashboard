Analytics Dashboard
A modern and responsive analytics dashboard built with React, Tailwind CSS, and Recharts.
The dashboard visualizes business metrics in real-time using interactive charts and KPI cards.
 Features
 Revenue Line Chart
Feature Usage Bar Chart
User Segmentation Pie Chart
KPI Cards
 Global Date Range Filter
 Automatic Data Refresh (Every 30 Seconds)
Loading Skeleton UI
Export Chart Data to CSV
Fully Responsive Layout
Clean Modern UI
